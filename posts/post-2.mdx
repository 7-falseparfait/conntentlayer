## What Are Hooks?

React Hooks let you use state and other React features without writing a class component. They're just functions that let you "hook into" React's functionality.

## useState - Managing State

The most common hook you'll use is `useState`. It lets you add state to functional components:

```javascript
const [count, setCount] = useState(0);
```
